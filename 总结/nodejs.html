 npm
官网： https://www.npmjs.com.cn/

更新npm：npm install npm@latest -g （npm@latest最新版本，也可以写指定版本例npm@2.1.1）

安装包--安装模块
npm install <packageName>  临时安装模块packageName
npm install <packageName> --global (-g) 全局安装模块packageName，任何目录下都可以访问（往系统注册了命令---全局安装的路径：C:\Users\asus\AppData\Roaming\npm）
npm install <packageName> --save （-S） 安装模块到项目依赖   ---- 项目上线需要的模块
npm install <packageName> --save-dev （-D）  安装模块到开发依赖  ---- 开发时模块，测试模块
npm install <packageName>@*** -D  指定版本安装某一个模块packageName

卸载
npm uninstall <packageName>

mongodb
安装mongodb环境 下载地址： https://www.mongodb.com/download-center#atlas

安装mongodb的可视化工具Robomongo：http://www.softpedia.com/get/Internet/Servers/Database-Utils/Robomongo.shtml#download

Mongodb使用：npm i mongodb@2 -D

配置mongodb：
1、在系统的根目录下，创建一个data，data下创建一个db目录，在data目录下创建一个文本文档，使用notepad++软件打开
2、另存为bat文件，文件名称为startMongo.bat
3、再继续另存为mongo.bat文件，删除新建的文本文档
4、使用notepad++打开那两个bat文件

startMongo.bat   -----  一定要注意使用自己的路径
c:
cd C:\Program Files\MongoDB\Server\3.4\bin
mongod --dbpath f:\data\db

mongodb.bat -----  一定要注意使用自己的路径
c:
cd C:\Program Files\MongoDB\Server\3.4\bin
mongo

每一次修改文件，都需要重新启动服务器，引入supervisor模块，进行服务器的热替换    
cnpm i supervisor -g
修改package.json中scripts选项
"dev": "supervisor ./bin/www"
 命令：cnpm run dev
